# Third-Party Licenses

Tweed downloads and uses the following tools at runtime. They are not bundled
with Tweed's source code but are fetched on first run if not already installed.

---

## scc (Sloc, Cloc and Code)

- **Repository:** https://github.com/boyter/scc
- **License:** MIT
- **Copyright:** Copyright (c) 2018 Ben Boyter

Used for code metrics: line counts, language detection, cyclomatic complexity,
and COCOMO estimates.

---

## Beret

- **Repository:** https://github.com/chapeaux/beret
- **License:** MIT
- **Copyright:** Copyright (c) 2026 Chapeaux

Used for structural AST analysis: function/class counts, test detection,
architecture layer identification, and call graph analysis.
